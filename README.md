## Install

### 1. Clone the source code or create new project.

```shell
git clone https://github.com/Carl-Javier/blog-exam.git
```

### 2. Install npm

```shell
npm install
```

### 3. Run command setup.sh

```shell
sh setup.sh
```

### 3. Use npm run dev

```shell
npm run dev
```
